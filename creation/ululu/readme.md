## ululu语速成教程

    o       我
    a       你
    le      爱

    ululu   表示肯定/表示开心/表示Let it be
    ulala   同ululu的含义一样，是年轻人较喜欢使用的惯用语

    ululu后面可跟任意多的lu, ulala后面可跟任意多的la, 用来自由表现心情

    叙述用平调，疑问用升调

    olea    我爱你
    olelelea        我爱爱爱你
    aleo    你爱我吗？


    一组简单对话:
    Daddy:  olelelea,  aleo?
    女儿:   ulalalalalalalala

    翻译过来就是：
    Daddy:  我爱爱爱你，  你爱我吗？
    女儿:   是呀是呀是呀是呀是呀是的呀~~~~~~~~
