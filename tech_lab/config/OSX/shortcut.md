符号-键映射表
⌘    Command 键
⌃    Control 键
⌥    Option 键
⇧    Shift 键
⇪    Caps Lock
fn    功能键

Command + Shift + 3     剪切全屏到一个新文件里(文件放在桌面上)
Command + Shift + 4     自定义剪切选定屏幕内容到桌面新文件里
上述两组合键分别加上control键，会剪切到系统剪贴板里。

alt + o = ø
alt + z = Ω
alt + v = √
